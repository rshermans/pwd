# 🛠️ Guia de Configuração: Repositório pwd

Este guia contém os dados necessários para configurar o repositório e o sistema de gestão de tarefas no GitHub.

## 1. Configuração do Repositório
**Nome do Repositório:** `pwd`
**Descrição:** Repositório central de acompanhamento e produção de artefactos para o projeto Portuguese Wallet Digital (Maio-Junho 2026).
**Visibilidade:** Público ou Privado (conforme preferência da equipa).

## 2. Configuração do GitHub Projects
Siga estes passos para criar o sistema de gestão de tarefas:

1. Vá ao separador **Projects** no seu perfil ou organização e clique em **New project**.
2. Selecione o template **Board** (ou comece um em branco).
3. **Colunas Recomendadas (Status):**
   - `🔘 Por Fazer` (Todo)
   - `🔵 Em Curso` (In Progress)
   - `👀 Em Revisão` (In Review)
   - `✅ Concluído` (Done)
   - `⏸️ Bloqueado` (Blocked)

## 3. Dados para Importação de Tarefas
Se for criar as tarefas manualmente ou via script, use estes campos baseados no seu planeamento:

| Campo | Opções / Valores |
|---|---|
| **Labels (Etiquetas)** | `Instagram`, `LinkedIn`, `WhatsApp`, `Webinar`, `Newsletter`, `Vídeo` |
| **Milestones (Marcos)** | `Fase 1: Preparação`, `Fase 2: Produção`, `Fase 3: Lançamento` |
| **Assignees** | Atribua as tarefas aos membros responsáveis da equipa |

## 4. Ligação de Artefactos
Para manter o repositório organizado, sempre que fechar uma tarefa (Issue):
1. Faça o upload do ficheiro para a pasta correspondente (ex: `conteudo/instagram/post-01.md`).
2. No comentário de fecho do Issue, coloque: *"Concluído. Artefacto disponível em: [link para o ficheiro]"*.

---
*Configuração sugerida para pwd v1.0*
