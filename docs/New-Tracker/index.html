Sobre o Projeto

Missão e Objetivos a Longo Prazo

O "Hub de Progresso"

Este espaço foi criado para democratizar o acesso ao processo de criação de conteúdo sobre Blockchain. Acreditamos que a transparência no desenvolvimento é a melhor forma de educar e envolver a comunidade.

Visão Disruptiva

Utilizar ferramentas de desenvolvimento como o GitHub para gerir e divulgar conteúdo não-técnico é um passo em direção a uma colaboração mais aberta e descentralizada.
